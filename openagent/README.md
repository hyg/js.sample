# OpenAgent - P2P Network Implementation

- openagent官网：https://openagents.org/
- 代码库：openagents-develop 文件夹

## 需求

- 开发openagent节点软件。
- 运行后生成唯一的agent ID。
- 节点软件按照decentralized模式连接，network名称可以配置。
- 如果发现其它节点显示相信信息。
- 可以与其它节点进行对话。在命令行接收用户输入，显示对话过程。

## 测试

- network名称： PSMD
- 退出时把详细日志保存在本文件夹下。
