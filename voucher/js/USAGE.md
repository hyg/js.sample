# 电子凭证自动整理工具使用说明

## 安装

确保已安装Node.js，然后在项目目录中运行：

```bash
npm install
```

## 使用方法

### 1. 批量处理模式

将电子凭证内容保存为文本文件，然后使用以下命令处理：

```bash
node voucher.js < your-voucher.txt
```

或者使用npm脚本：

```bash
npm start < your-voucher.txt
```

### 2. 交互式模式

运行交互式模式，可以连续处理多个电子凭证：

```bash
node voucher-interactive.js
```

或者使用npm脚本：

```bash
npm run interactive
```

在交互式模式下：
1. 输入电子凭证的多行内容
2. 输入"END"结束当前凭证的输入
3. 系统会自动生成AVR.id.yaml文件
4. 继续输入下一个电子凭证或按Ctrl+C退出

## 输出文件

生成的YAML文件将保存在项目目录中，文件名为`AVR.{id}.yaml`，其中id是自动递增的数字。
